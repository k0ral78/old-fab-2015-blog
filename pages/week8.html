Computer-controlled machining (Mar 18)

This week the assignment was very time consuming for me. I need to practice a lot with CAD programs. I decided to use openSCAD again because I really want to practice with more complicated code using modules. I also wanted to try parametric way of thinking. I never used CAD programs before Fab Academy then every time I have to design something I know it will be time consuming. Together with my local instructor I decided to design a standing desk for the Modela MDX-20 and a laptop in order to have a (((POSTAZIONE))) only for this :) I found some difficulties exporting the file from openSCAD to .dxf but then I solved the problem exporting the .stl file and exporting the .dxf with FreeCAD. Before using the (FRESA) I wanted to test my design with laser cutter. In this way I also completed (????) the assignment of laser cutting because I still had to do it.
