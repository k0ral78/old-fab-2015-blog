Website

I always wanted to be able to create a simple website. Fab Academy has been the right personal motivation. I did a simple website with the essential information but I plan to expand it during this semester. I would have more time to make my blog nicer, but during this semester I’ll definitely do it. I always think how to do my things better. Usually when i finish doing my job i start thinking how to do better the same thing. Here below some of useful tips i found to understand website tools.

  • HTML5 and CSS: i have done this tutorial and i found it very useful.
  • I found some difficulties installing Jekyll on Linux Mint, but after installing ruby-full following these instructions i solved it.
  • This has been a very useful tutorial to understand jekyll step-by-step
  • Bootstrap: at first I was a little bit confused but I found this tutorial and it was very useful. It is a step-by-step guide and there are many little interactive code examples.

Something about project ideas

I found some difficulties about choosing my final project just because i’d like to do many things :)
Here it is the list of the projects I thought:

  • Auto sat finder tracking: a platform on which you can put a dish antenna and automatically it point to the right satellite.
  • Portable radio bridge: I thought about something very light to carry. A mountable light trellis made of wood, something like a kit. Eventually with motors to help to lift up higher. Obviously an auto point system with the other far end of radio bridge.
  • A ground vehicle, a mobile platform which is able to go even in the worst countryside. Eventually autonomous with an auto “go back home” system when battery goes low. Obviously it has to stream video in real-time and you also can drive it remotely.
  • A wireless rechargeable system: I like the idea to go in depth on induction recharge.
  • I also like the world of wireless sensor networks and IoT, even because I studied something for one year. Regarding this idea I hope that with the project i chose I will have more knowledge to be able to do a wearable underwater network. It could be useful to monitor scuba divers during immersion.
  • I also would have liked to do some toys for children. I think that it is very funny and I’d like to do a funny toy for my nephew :) Anyway, my project could also be even a very funny toy :)
  • I liked the pick and place machine for SMD electronics components. I think one day i’ll try to do it.
  • In the end, i wanted to do a video door phone for my parent’s house. In the last months I am a little bit worried about thefts in houses. Furthermore I’d like to go in depth in home security systems which can give a feeling of more safety. For this reason I installed some IP cam in my house but we still don’t have a video door phone.
